
# MakeShop E-commerce

MakeShop is a responsive React e-commerce single-page application that allows users to browse products, view product details, manage a shopping cart, and go through a checkout flow.

## Live Demo

https://makeshop.onrender.com

## Repository

https://github.com/alizonekreate/Makeshop

## Tech Stack

- React.js
- React Router
- JavaScript
- HTML5
- CSS3
- Render

## Features

- Responsive e-commerce website layout
- Dynamic product pages
- Shopping cart functionality
- Product quantity and cart management
- Checkout flow
- Client-side routing using React Router
- Reusable React components
- Clean and user-friendly interface

## Project Purpose

This project was built to practice creating a real-world React single-page application with routing, reusable components, and interactive e-commerce features. It helped me strengthen my understanding of component structure, state management, user experience, and frontend application flow.

## What I Learned

- Structuring a React single-page application
- Creating reusable UI components
- Implementing page navigation with React Router
- Managing cart-related user interactions
- Building responsive layouts for different screen sizes
- Improving user experience through organized product and checkout flows
